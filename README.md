# QUIK

[![Manual Release](https://github.com/octoshrimpy/quik/actions/workflows/manual-release.yml/badge.svg)](https://github.com/octoshrimpy/quik/actions/workflows/manual-release.yml)
<a href="https://hosted.weblate.org/engage/quik/">
<img src="https://hosted.weblate.org/widget/quik/svg-badge.svg" alt="Translation status" />
</a>

QUIK to otwartoźródłowy (open source) zamiennik [domyślnej aplikacji do obsługi wiadomości](https://github.com/android/platform_packages_apps_mms) w systemie Android. Jest to kontynuacja projektu [QKSMS](https://github.com/moezbhatti/qksms).

## Funkcje

- Planowanie wiadomości
- Kopia zapasowa wiadomości
- Zintegrowane w aplikacji rozpoznawanie mowy na tekst i tekstu na mowę
- Blokowanie i archiwizowanie wiadomości
- Wiadomości głosowe
- Załączniki dowolnego typu pliku
- Sortowanie wiadomości
- Przypinanie wiadomości
- Opóźnione wysyłanie
- Szybka odpowiedź z poziomu powiadomień
- Akcje przypisane do gestów przeciągnięcia
- Obsługa reakcji emoji
- i wiele, wiele więcej

## Zgłaszanie błędów

Świetne zgłoszenie błędu zawiera opis problemu oraz kroki potrzebne do jego odtworzenia. Musimy wiedzieć, czego szukamy i gdzie tego szukać.

Zgłaszając błąd, prosimy o podanie następujących informacji:
- Kroki do odtworzenia problemu
- Wersja QUIK
- Informacje o urządzeniu / systemie operacyjnym
